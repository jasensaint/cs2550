Project Design

Model

Constants
Easy< >, Medium< >, Hard< > pregenerated boards, and numbers 1-9

Game Setup

difficulty()
This alllows you to pick a difficulty of game.

startGame()
This will populate the numbers and start the game timer.

reset()
clears game board and resets the game timer.

Timer

This is a simple timer that will be called by startGame(), reset by reset() and stopped by successful completion.
resetClock()
resets the clock to 0's

stopClock()
stops clock and keeps the final value displayed.

startClock()
Starts the clock.

Play

enterNumber()
This will populate the number and then call the completion check.

complete()
This will check if the game is complete.
