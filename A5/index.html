Sudoku

Sudoku: Is a simple puzzle game that has been arround for centuries. As with many puzzle games it takes a few moments to learn and life time to master. The idea behind the game is to fill a 9x9 grid with the numbers 1-9 without any of numbers overlapping in their areas. This takes math and patern analysis skills. It may be harder than it looks.

Rules

    1. The grid board will start with some numbers already filled in.
    2. Each row must have all of the numbers 1-9 without any duplicates.
    3. Each column must have all of the numbers 1-9 without any duplicates.
    4. Each 3x3 section must have all of the numbers 1-9 without any duplicates.
    5. Add numbers to the grid until it is full while abiding by the stated rules.
